使用指南

步骤一：下载插件

首先，您需要下载我们的插件。点击下方链接即可下载插件。

下载插件

步骤二：安装插件

下载完成后，您需要将插件安装到您的浏览器中。具体步骤如下：

    对于 Chrome 浏览器：

    1. 打开您的 Chrome 浏览器，并进入扩展程序页面（chrome://extensions/）；
    2. 在页面右上角开启“开发者模式”；
    3. 点击“加载已解压的扩展程序”按钮，选择您下载的插件所在的文件夹；
    4. 安装成功后，您可以在浏览器的扩展程序栏中看到您的插件图标。

    对于 Edge 浏览器：

    1. 打开您的 Edge 浏览器，并进入扩展程序页面（edge://extensions/）；
    2. 开启“开发者模式”；
    3. 点击“加载已解压的扩展程序”按钮，选择您下载的插件所在的文件夹；
    4. 安装成功后，您可以在浏览器的扩展程序栏中看到您的插件图标。

    对于 Firefox 浏览器：

    1. 打开您的 Firefox 浏览器，并进入附加组件页面；
    2. 点击“齿轮”图标，选择“调试附加组件”；
    3. 点击“临时载入附加组件”按钮，选择您下载的插件所在的文件夹，选取manifest.json文件；
    4. 安装成功后，您可以在浏览器的附加组件栏中看到您的插件图标。

步骤三：使用插件

安装完成后，您可以开始使用我们的插件了。具体使用方法如下：

一、翻译设置

  1. 在网页右上角找到扩展按钮，打开插件
  2. 点击齿轮状的设置按钮进入设置界面
  3. 配置API端点以及api_key
  4. 点击“保存设置”保存

二、翻译网页以及恢复网页的使用说明

  1. 首先，打开您需要翻译的网页页面；
  2. 在网页右上角找到扩展按钮，打开插件；
  3. 在翻译窗口中选择您需要翻译的目标语言，源语言可以自动识别，也可以手动选择；
  4. 点击“翻译网页”按钮，即可将网页翻译为所需语言；
  5. 点击“恢复网页”按钮，即可将网页还原为原网页。

三、划词翻译的使用说明

  1. 在网页用鼠标中选取需要翻译的文本；
  2. 右键鼠标打开菜单选项；
  3. 选择翻译的目标语言；
  4. 点击菜单项中的语言选项即可直接翻译所选文本。

版本说明

本插件支持以下版本的浏览器或浏览器内核：

  • Chromium 88 及以上版本，包括chrome浏览器及其他支持chromium内核的浏览器（如360浏览器、360极速浏览器）
  • Edge浏览器，内核 88 及以上版本
  • Firefox浏览器，内核 102 及以上版本
